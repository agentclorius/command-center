OpenClaw til Webudviklere

Byg websites 3-4x hurtigere med en AI-agent der kører lokalt på din Mac

🤖 Hvad er OpenClaw?

En AI-agent der kører lokalt på din computer. Den kan bygge hele websites fra en kort brief, iterere i realtid, og deploye — mens du snakker med den via Telegram fra din telefon.

🏗️

Bygger hele sites

Fra brief til færdig side. Astro, Next.js, HTML — hvad du vil.

🔄

Itererer i realtid

"Flyt headeren, skift farve, tilføj sektion" — 30 sek pr. ændring.

🚀

Deployer for dig

GitHub Pages, Vercel, Netlify — ét kommando.

📱

Telegram-styret

Skriv fra telefonen. Agenten arbejder på din Mac.

⚡ Workflow: Brief → Færdig Side

1
Kunden sender brief → du videresender til din agent via Telegram
2
Agenten bygger et udkast (Astro + Tailwind, Next.js, plain HTML)
3
Du reviewer i browseren — live preview link
4
"Gør headeren mørkere, tilføj testimonials, skift til dansk"
5
Agenten retter → du godkender → deploy 🚀
Tidsbesparelse: Et simpelt firmasite der normalt tager 1-2 dage → 1-2 timer med iteration.

🎬 Live Demo: Byg et site på 5 min

1. Giv agenten en opgave

"Byg en landingpage for en tømrermester i Aalborg. Firmanavn: Hansen Tømrer. Services: nybyg, renovering, tagarbejde. Farver: mørkeblå og hvid. Hero, services-sektion, om os, og kontaktformular. Brug Astro + Tailwind."

2. Agenten arbejder

# Agenten gør automatisk:
→ Opretter projekt-mappe
→ Installerer dependencies
→ Bygger alle komponenter
→ Starter dev server
# Du ser resultatet på localhost

3. Iterér via Telegram

"Tilføj en galleri-sektion med placeholder-billeder"
"Skift fonten til Inter"
"Gør CTA-knappen grøn i stedet for blå"
"Tilføj Google Maps embed i kontakt-sektionen"

Hver ændring tager 30-60 sekunder.

4. Deploy

"Deploy til GitHub Pages"

Færdig. Kunden har en live URL.

🛠️ Setup (5 minutter)

# 1. Installer
npm install -g openclaw

# 2. Init
openclaw init

# 3. Tilslut Telegram
openclaw channel add telegram

# 4. Klar! Skriv til din bot i Telegram 🚀

💡 Pro Tips

  • Vær specifik i din brief — jo mere kontekst, jo bedre første udkast
  • Brug screenshots — "Gør det mere som denne side"
  • Gem templates — byg én gang, genbrug til lignende kunder
  • Sæt regler op — "Brug altid Astro + Tailwind, dansk default, Inter font"
  • Coding agent — til større projekter, spawn en dedikeret agent
