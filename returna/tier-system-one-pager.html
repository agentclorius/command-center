Category 1

Basic Processing

TRC Level 1 — Foundation
Core ITAD processing: receive, erase, sort, recycle.
Required Certifications
  • ISO 27001 (or parent scope coverage)
  • Returna-approved erasure tool
  • ISO 14001 + WEEE registration
Capabilities
  • Secure intake & asset registration
  • Data erasure (standard devices)
  • Basic sorting & grading
  • WEEE-compliant recycling
  • Secure storage & chain-of-custody
Eligible Work
Recycling Basic ITAD Secure disposal
Category 2

Diagnostics & Grading

TRC Level 2 — Intermediate
Advanced processing with diagnostics, testing, and value recovery.
Required (Cat 1 + These)
  • Advanced data erasure capability
  • Hardware diagnostics tooling
  • ISO 9001 (recommended)
Capabilities
  • Everything in Cat 1, plus:
  • Advanced erasure (servers, mobile, NVMe)
  • Full hardware diagnostics
  • Functional testing & cosmetic grading
  • Component harvesting
  • Resale-ready preparation
Eligible Work
All Cat 1 work Value recovery Enterprise ITAD Resale prep
Category 3

Full Refurb & Box Programs

TRC Level 3 — Advanced
Full lifecycle: box programs, refurbishment, redeployment, white-glove.
Required (Cat 2 + These)
  • TRC Level 3 certification (or equivalent)
  • ISO 9001
  • Box program logistics capability
  • Resale channels (B2B / marketplace)
Capabilities
  • Everything in Cat 2, plus:
  • Enterprise box programs (home pickup/drop-off)
  • High-value refurbishment (skins, keys, screens)
  • OS imaging & MDM enrollment
  • Kitting & white-glove delivery
  • Multi-country logistics coordination
Eligible Work
All Cat 1+2 work Box programs Refurbishment Redeployment White-glove

  • Level 2 — Advanced: Keyboard replacement, component-level work
  • Level 3 — Screen Specialist: Display repair, advanced diagnostics
  • Level 4 — Master Refurbisher: Full lifecycle refurbishment

TRC supplies skins, SticKeys, tools. Returna can route partners to TRC for training.
