# Return Hub Partner Scoring System

_Last updated: 2026-02-05_

## Strategic Context

Returna is building a neutral orchestration platform. We need Return Hubs that:
- **Need us** — can't generate enterprise volume alone
- **Complement us** — don't have competing software/marketplace
- **Trust us** — see us as enabler, not competitor

**Critical:** Returna founders are divesting from Scandic IT to ensure platform neutrality. No ITAD ownership = no conflict of interest.

---

## Ideal Return Hub Profile

| Attribute | Ideal | Acceptable | Disqualify |
|-----------|-------|------------|------------|
| **Size** | Small (10-50 employees) | Medium (50-200) | Large (200+) |
| **Revenue** | €1-10M | €10-30M | €30M+ |
| **Geographic Scope** | Local/Regional | National | Multi-national network |
| **Own Software** | None | Basic inventory | Full asset management platform |
| **Own Marketplace** | None | Small B2B resale | Large marketplace/retail |
| **Certifications** | ISO 27001 + 14001 | At least one ISO | None |
| **Enterprise Clients** | Few/None | Some | Many (don't need us) |

---

## Scoring Rubric (0-100 points)

### 1. Size & Independence (30 points max)

| Criteria | Points |
|----------|--------|
| Small company (10-50 employees) | 30 |
| Medium company (50-200 employees) | 20 |
| Large company (200+ employees) | 0 |
| Part of larger corporate group | -10 |

### 2. Certifications (25 points max)

| Criteria | Points |
|----------|--------|
| ISO 27001 (data security) | 10 |
| ISO 14001 (environmental) | 8 |
| ISO 9001 (quality) | 5 |
| R2 Certified | 8 |
| e-Stewards Certified | 8 |
| ADISA Certified | 8 |
| WEEE Compliant | 5 |
| No certifications | -15 |

*Cap at 25 points — having more certs is good but diminishing returns*

### 3. Technology Gap (20 points max)

| Criteria | Points |
|----------|--------|
| No asset management software | 20 |
| Basic spreadsheet/inventory only | 15 |
| Simple software, open to switch | 10 |
| Established software, unlikely to switch | 0 |
| Own platform competing with Returna | -20 |

### 4. Market Position (15 points max)

| Criteria | Points |
|----------|--------|
| Local/city focus | 15 |
| Regional focus (1-2 countries) | 12 |
| National focus (single country) | 8 |
| Multi-national operations | 0 |
| Global ITAD network | -15 |

### 5. Processing Capabilities (10 points max)

| Criteria | Points |
|----------|--------|
| Full ITAD processing (erasure, diagnostics, grading) | 10 |
| Partial processing (needs partners for some) | 7 |
| Basic intake/logistics only | 4 |
| Collection only (no processing) | 2 |

---

## Score Interpretation

| Score | Rating | Action |
|-------|--------|--------|
| 80-100 | 🟢 **Excellent Fit** | Priority outreach |
| 60-79 | 🟡 **Good Fit** | Standard outreach |
| 40-59 | 🟠 **Potential Fit** | Research more before contact |
| 20-39 | 🔴 **Poor Fit** | Likely not a match |
| <20 | ⚫ **Disqualify** | Do not pursue |

---

## Disqualifiers (Automatic Reject)

- [ ] Revenue > €50M
- [ ] Own asset management platform (competing product)
- [ ] Part of global ITAD network (Foxway, Ingram, Iron Mountain, etc.)
- [ ] No ISO certifications and unwilling to get them
- [ ] Only does recycling (no refurb/resale capability)
- [ ] Bad reputation / legal issues
